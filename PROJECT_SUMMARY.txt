   ✓ Manifest: Valid JSON (no errors)
   ✓ Skills: All 3 SKILL.md files present
   ✓ Icons: Both PNG files in correct location
   ✓ Structure: Correct folder hierarchy
   
   Commands to run:
   ```
   node -e "console.log(JSON.parse(require('fs').readFileSync('manifest.json')))"
   ls -la skills/*/SKILL.md
   file color.png outline.png
   ```

### STEP 3: Package Plugin (2 min)
   Choose ONE packaging method:
   
   Option A - PowerShell (Windows):
   ```powershell
   Compress-Archive -Path manifest.json, color.png, outline.png, skills `
     -DestinationPath copilot-dev-camp.zip -Force
   ```
   
   Option B - npm:
   ```bash
   npm run package          # Windows
   npm run package:unix     # macOS/Linux
   ```
   
   Option C - Bash (macOS/Linux):
   ```bash
   zip -r copilot-dev-camp.zip manifest.json color.png outline.png skills/
   ```

### STEP 4: Test Locally (5 min)
   ```bash
   npm install -g @microsoft/m365agentstoolkit-cli
   atk auth login
   atk install --file-path "./copilot-dev-camp.zip" --scope Personal
   ```
   
   Verify in Cowork:
   ✓ Open Copilot → Cowork → Sources & Skills
   ✓ See "Copilot Dev Camp" plugin
   ✓ See all 3 skills listed

### STEP 5: Deploy to Tenant (5-30 min)
   Choose deployment path:
   
   Path A - Personal (uses atk CLI)
   Path B - Tenant Admin (M365 Admin Center)
   Path C - Intune (Enterprise deployment)
   
   See DEPLOYMENT.md for detailed instructions
